# SocialMoon Agent Instructions

## Core Rule: Tone 3 (Direct and Honest)
All website copy must follow this voice:
- No fluff, no hype, no fake urgency
- Straight language like a real conversation
- Facts first, promises second
- Confidence without arrogance

Use lines like:
- "Social media takes work. Let us do it."
- "No magic. No shortcuts. Just real strategy."
- "If something is not working, we change it."

Never use lines like:
- "Go viral in 30 days"
- "Guaranteed 10x growth"
- "Set it and forget it"

## Truth and Verification Policy
Do not publish any claim that cannot be proven.

Allowed only when verified:
- Client counts
- Growth percentages
- Revenue, leads, reach, engagement numbers
- Awards, certifications, rankings
- Partner logos, client logos, brand names
- Testimonials, reviews, ratings

If proof is missing:
1. Remove the claim, or
2. Replace with a non-numeric factual statement

If a request needs unknown details (example: client name, stat, award, timeline), ask the user first before adding it.

## Positioning Rules
SocialMoon is a social media marketing agency.

Default language:
- clients, brands, businesses
- strategy, content, campaigns, execution
- testing, iteration, reporting

Avoid SaaS/product framing unless the user explicitly confirms it:
- platform, dashboard, product suite
- automation-only messaging
- technical product names that sound fabricated

## Copywriting Standards
Every section should be:
- specific
- measurable only when verified
- easy to understand
- free of corporate filler

Preferred style:
- short sentences
- active voice
- concrete outcomes

Avoid:
- vague superlatives ("best", "world-class", "industry-leading")
- guaranteed outcomes
- inflated timelines

## Content Safety Checklist (Run Before Finalizing)
For each page, verify:
1. Any number is sourced or user-approved
2. Any company/client name is real and user-approved
3. Any testimonial is real and approved
4. No fake dashboard/demo data is shown as real
5. Tone matches Direct and Honest

If any check fails, remove or rewrite.

## Agent Behavior Rules
- Never invent facts to "fill" content
- Never add random company names, metrics, or achievements
- Ask the user before adding unverifiable details
- When unsure, choose plain truthful copy over bold claims

## Default Rewrite Pattern
When copy is weak or exaggerated, rewrite using:
1. What we do
2. How we do it
3. What result we aim for (without fake guarantees)

Example:
- Bad: "We guarantee explosive growth in weeks."
- Good: "We plan, publish, test, and improve every month to grow your social presence steadily."
