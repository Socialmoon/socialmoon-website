# Agent Instructions: Remove Fake/Unnecessary Stats

## Objective
Remove inflated, unverified, or misleading statistics from the SocialMoon website that sound fake or contribute to misleading marketing claims.

## Scope
This task applies to all pages on the website, starting with the home page.

## Criteria for "Fake" Stats
Stats should be removed if they meet any of these criteria:
1. **Unverified numbers** - Claims without concrete data backing them (e.g., "10,000+ users" with no actual proof)
2. **Suspiciously high percentages** - Numbers like "98% satisfaction rate" or "100% match" that are unrealistic
3. **Generic vanity metrics** - Hollow claims like "24/7 support available" that don't demonstrate actual value
4. **Inflated performance claims** - Exaggerated results like "+340% reach", "+214% pipeline growth" without real case data
5. **Demo/mockup data presented as real** - Test numbers shown in dashboards presented as actual performance
6. **Too-round numbers** - Suspiciously clean numbers like "500K+ posts managed" or "50+ businesses"

## SaaS-Specific Language to Remove
Since SocialMoon is a **social media marketing agency** (not a SaaS company), remove or rephrase:
- **"Platform"** terminology (e.g., "Our platform", "dashboard", "cloud optimization")
- **Technical jargon** - "WorkflowOS", "CloudTrim", "OpsFlow AI" sound like SaaS products, not agency services
- **Automation-heavy claims** - Agency focuses on human expertise + strategic marketing, not pure automation
- **Enterprise software messaging** - Systems panels, live dashboards, API references
- **"Users" terminology** - Should be "clients" or "businesses we serve"
- **Subscription/billing language** - Remove any pricing tier suggestions
- **"AI-powered" vagueness** - If mentioning AI, be specific about how it's used in marketing context
- **Product comparisons** - Avoid comparing to other SaaS tools

## Language to Use Instead
- ✅ "Marketing agency" / "Social media marketing experts"
- ✅ "Clients" / "brands" / "businesses we work with"
- ✅ "Our services" / "Our solutions" / "What we offer"
- ✅ "Strategy" / "Campaign" / "Content creation"
- ✅ "Team" / "Team support" / "Expert consultation"
- ✅ "Results" / "Case studies" / "Client success stories"

## SaaS Visuals & Components to Remove/Rephrase
- ❌ **Dashboard mockups** showing fake metrics (the "Performance Overview" section with fake stats)
- ❌ **"Systems running" panels** with technical-looking code editors - REPLACE WITH better service display
- ⚠️ **Service names are CORRECT** (Content & Social Growth, Lead Generation, OpsFlow AI, etc.) - Keep them but display better
- ❌ **Fake metrics in system panels** ("+340% reach", "57 leads/mo", "-60% manual work", etc.)
- ❌ **Multiple product/tier suggestions** - Stick to core service offerings
- ✅ **Replace with:** Better content about actual services, portfolio examples, client testimonials, case studies

## Action Steps for Each Page
1. Identify all statistics displayed on the page
2. Evaluate each stat against the "Fake Stats" criteria
3. **Remove completely** if clearly fake
4. **Replace with honest metrics** if the stat is important for credibility:
   - Use real company numbers if available
   - Replace with qualitative benefits instead
   - Use realistic, lower numbers if you have them
5. **Simplify claims** - Convert braggadocio to humble, honest language
6. **Replace SaaS language** with agency-focused messaging
7. **Remove technical mockups** - Replace with actual portfolio/case study content
